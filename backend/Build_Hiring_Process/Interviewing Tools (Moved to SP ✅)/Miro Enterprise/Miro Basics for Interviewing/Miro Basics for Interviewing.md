# Overview

Miro is a collaboration tool that has several components enabling real\-time collaboration between participants. This page provides a quick\-hit of elements to focus on in order to help candidates and interviewers use this tool effectively. 

# Miro Window

Miro offers an effectively infinite wall space to draw on. In addition to the main drawing area. 

There are 5 palette areas:

* Toolbox
* Undo/Export
* Participants \& Sharing
* Navigation
* Interaction
# The Toolbox

Focus your attention on the tool box – There are only a handful of tools you’ll need:

* Primary Tools:


	+ Text Tool
	+ Shape Tool
	+ Connection Line Tool
* Secondary Tools:


	+ Stickie Note
	+ Highligher
# Icon Sets

For certain scenarios, you may want to use specific icon sets for particular cloud providers. Click the ellipsis at the bottom of the toolbox and select the appropriate icon set form the Apps dialog.
