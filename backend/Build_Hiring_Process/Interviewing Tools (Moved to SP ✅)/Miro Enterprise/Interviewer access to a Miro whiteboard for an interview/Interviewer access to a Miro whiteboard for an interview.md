# Overview

Using Miro is very straight forward: You simply click on the link to the whiteboard and you will be added as a collaborator on the whiteboard. You will follow this process to access a Miro whiteboard for an interview:

* Obtain a link to the whiteboard being used
* Click on the link
* Start Collaborating!

noteAnonymously shared Miro whiteboards are accessible by anyone with access to the link for the whiteboard. For this reason, you should ensure that no PII, Proprietary or Confidential information is added to a Miro whiteboard within your interview.

### Who am I?

If you open a shared Miro Lite whiteboard link in a browser session where there **is not** a Miro auth token stored (meaning you **have not** logged into Miro Enterprise), then you will be automatically assigned a ‘Guest User’ name that is unique on the whiteboard.

If you open a shared Miro Lite whiteboard link in a browser where there **is** a Miro auth token stored (meaning you **have** logged into Miro Enterprise), then you will be recognized as your Miro user name.
