# Making the Most of Miro: Tips & Tricks from the Field

Miro is a great for collaboration and whiteboarding. Is it the perfect tool for coding interviews? Nope. Here are some tips \& workarounds to bridge some gaps until a more robust tool is in place. 



---

## Preparing for your Session

* Set up new board for each session with the coding question(s) you plan to have the candidate work, framed out. 


	+ **Tip**: Samples from Miro are formatted and ready to go, use these to make life easy!  
	[Miro: Coding Samples](https://miro.com/app/board/o9J_lDwySeY=/)
* Verify each box you intend to be used for code entry is a **Text** box. 


	+ **Tip**: Enter some characters to be sure the text is readable.



---

## Creating New Scenarios

### Use Text Boxes

* Make sure to use **Text** boxes where you want the candidate to write code. As Miro’s documentation suggests, there are 2 ways to add text to the board:


	+ Select **Text** on the toolbar or [press T on your keyboard](https://help.miro.com/hc/en-us/articles/360017731033), click anywhere on the board, and start typing
	+ Drag and drop a text widget from the toolbar right to the board

**Sizing** 

You can **expand** text boxes 3 ways: vertically, horizontally, and diagonally.

* **Vertically**: To vertically resize text boxed in Miro, enter new\-line characters (i.e. press **Return/Enter** a few times inside the text box) and the box will grow in height. Using this method, text size remains constant, box changes height.
* **Horizontally**: Simply drag the text box horizontally, text size remains constant, box changes width.
* **Diagonally**: Dragging the text box diagonally scales text size only, relative size of box remains constant.

**Styling**

To apply a particular **style** from one text box to another, you **copy** and **paste** **style**. 

* Use the shortcuts **Ctrl \+ Alt\+ C** and **Ctrl \+ Alt \+ V** (*for Windows*) or **Cmd \+ Alt\+ C** and **Cmd \+ Alt \+ V** (*for Mac*)
* Or click the three dots on the text context menu and choose the option.

* To apply style to ALL text fields at the same time, select all objects on the board (you can use **Ctrl \+ A** *for Windows* or **Cmd \+ A** *for Mac*) and change the formatting.

Preferred font styling: 

* **font\-size**: 12 \- 16 pts
* **font**: Roboto Mono
* **color**: Black

Learn more about Miro’s **Text** features: [https://help.miro.com/hc/en\-us/articles/360017572094\-Text](https://help.miro.com/hc/en-us/articles/360017572094-Text)

## Known Gaps

### Miro doesn’t allow compiling/running code

* There is no ability to run or test code


	+ There’s no syntax highlighting
	+ There is no ability to link to or run db queries

### Miro’s customization is limited

* Tabs are single space.
